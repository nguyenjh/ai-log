Commit tagging quick guide
[ai:gen]
AI generated the code; you reviewed & accepted it
[ai:assist]
You wrote it; AI helped debug, explain, or improve
[ai:doc]
AI helped write or improve docs, comments, markdown
[ai:test]
AI helped write or improve tests
[ai:plan]
AI used for planning, requirements, or architecture
[ai:skip]
No AI used in this commit
Example Commit Messages
Add note search functionality [ai:gen]
Fix tag filter bug — off-by-one in array slice [ai:assist]
Write ARCHITECTURE.md decision section [ai:doc]
Add unit tests for NoteStore [ai:test]
Discuss auth approach with team + Claude [ai:plan]

You do not need to tag every single commit if you are unsure. When in doubt, use [ai:assist]. The goal is honest signal, not perfect accounting.
